Hello, I'm Madhumitha.

I'm a mid-level software engineer with 4.5 years of experience specialising in cross-platform mobile development with Flutter/Dart and data solutions with SQLite and MySQL. I bridge technical delivery with strategic business insight.

04
Java · IBM Cloud · IoT

SilverCheetah — SCDF × IBM Hackathon

During the SCDF × IBM Call for Code hackathon, my team developed a Rapid-Response Emergency Detection System to address the critical need for elderly safety: the fact that falls often go undetected when a victim is too disoriented to call for help. Using IBM Cloud AI and IoT, we built a system that streams real-time data from wearable sensors to a cloud-based AI model designed to distinguish between everyday movements and actual accidents allowing a drastic reduction in response latency by automatically alerting emergency services with precise location and severity data. By moving from passive monitoring to proactive detection, the project demonstrates how we can use data and AI as a practical expression of care, ensuring that vulnerable individuals are never left invisible during a crisis.
